wp-nginx-http2

- Step.1) 基本となる WordPress 環境を構築
- Step.2) WordPress を日本語化対応させる
- Step.3) WordPress テーマを開発できるようにする
- Step.4) 任意の WordPress プラグインがインストールされた状態で起動する
